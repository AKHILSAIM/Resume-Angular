# Resume Web App

A full-stack Resume Web Application built using:

- Angular (Frontend)
- Node.js + Express (Backend)
- MySQL (Database)

## Features

- Enter resume details
- Store in MySQL database
- Retrieve and display resume
- Separate pages for form and resume view
- Clean resume layout matching PDF design

## Technologies Used

- Angular
- TypeScript
- Node.js
- Express.js
- MySQL

## How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
ng serve

Open http://localhost:4200
